# LOOP Detector Board

این برد عیناً از روی شماتیک قدیمی `main loop pcb3.SchDoc` (PDF ضمیمه) بازسازی شده.
هیچ تغییر منطقی نسبت به مدار اصلی اعمال نشده — فقط اتصالات مودم و MCU به جای
hard-wire، از طریق هدر backplane منتقل شده.

## بلوک‌های مدار

### 4 کانال ورودی حلقه (loop input chain)
هر کانال (`11`, `12`, `13`, `14`):
1. ورودی از `P1.x` (هدر 8-پین، یک سیم لوپ)
2. **R##A** (0Ω سری) + **C##B** (47nF به GND) → فیلتر ورودی
3. **شبکهٔ حفاظت ESD/spike**: 3× 2N7000 (Q##A, Q##B, Q##C) — کلامپ به GND/VCC
4. **C##A** (47nF اضافی)
5. **D##A** (1N5819 شاتکی) → کلامپ به GND
6. **R##C** (330K pull-up به VCC) + **R##B** (10K) + **R##D** (3.3K) → تقسیم‌کنندهٔ آستانه
7. **U50/x** (یک ربع از LM339) → مقایسه‌گر اصلی (open-collector خروجی)
8. **R##E** (1K سری gate)
9. **Q##D** (2SJ196 یا BSS92 P-channel) → سوییچ بار
10. خروجی به `pl.x`

### 2 مالتی‌پلکسر آنالوگ
- **U51 (HCF4051BEY)**: کانال‌های 0..7 — سیگنال‌های اولیه از LM339‌ها
- **U52 (HCF4051BEY)**: کانال‌های 0..7 — سیگنال‌های ثانویه (مثلاً برای ADC)
- A, B, C: address lines (از MCU)
- INH: enable (active-low از MCU)
- IO_COM: تک سیم برگشتی به MCU

### تنظیم آستانه (Threshold)
ولتاژ آستانه LM339 با `R##C / (R##C + R##B + R##D)` تنظیم می‌شه:
- VCC = 5V
- آستانه ≈ 5 × 330K / (330K + 10K + 3.3K) ≈ **4.81V** (high threshold)
- یا اگر input pull از pin – آستانهٔ پایین ≈ 5 × 3.3K / (10K + 3.3K) ≈ **1.24V** (low threshold)
- این نشون می‌ده مقایسه‌گر hysteresis داره (window comparator)

## ابعاد PCB پیشنهادی
80 × 80 mm، 2-layer. پیشنهاد:
- ورودی‌ها (P1) لب چپ
- خروجی‌ها (pl) لب راست
- LM339 و muxها وسط
- 2N7000 و 2SJ196 نزدیک هدرها (مسیر سیگنال کوتاه)
- GND plane در لایهٔ پایین

## تست
1. به یک کانال (مثلاً P1.1) اعمال 5V → باید Q11D روشن بشه و pl.1 ≈ VCC
2. اعمال 0V (یا open) → Q11D خاموش، pl.1 ≈ 0V
3. اسکن MUX: تغییر A,B,C باعث دیدن کانال‌های مختلف روی IO_COM می‌شه
